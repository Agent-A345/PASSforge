## 🔐 PASSforge — Secure Password Generator

A powerful and visually immersive Password Generator web app built using HTML, CSS, and JavaScript, designed to create highly secure passwords with entropy analysis, strength indicators, and passphrase generation.

## 🚀 Features
* **Cryptographically Secure Generation** – Uses crypto.getRandomValues() for true randomness.
* **Password Length Control** – Adjustable length (8–32 characters).
* **Passphrase Mode** – Generate memorable Diceware-style passwords (e.g., apple-river-stone).
* **Strength Meter** – Real-time password strength classification (Weak, Medium, Strong).
* **Entropy Analysis** – Displays entropy (in bits) and visual security level.
* **Crack Time Estimation** – Estimates how long it would take to brute-force the password.
* **Password History** – Stores recently generated passwords locally.
* **Copy & Reveal Options** – Toggle visibility and copy password securely.
* **Smart Mode** – Auto-generates highly secure passwords with optimal settings.

## 🎮 Controls
**Password Generation**
* Generate Button → Create password

**Advanced Controls**
* Passphrase Mode → Switch to word-based passwords
* Smart Mode → Auto-select secure configuration
* Reveal Button → Show/Hide password
* Copy Button → Copy to clipboard
